# 高中现代文练习平台

一个基于React和Express的高中现代文练习平台，参考Apple.com风格设计，支持按题目类型分类练习和智能作文批改。

## 功能特点

- 📚 **专题分类**：自动从题目数据中提取类型作为分类，包括语言赏析、表达效果、作用分析等
- 📝 **智能批改**：集成DeepSeek API进行作文智能批改，提供详细的评分和反馈
- 🎨 **现代设计**：参考Apple.com风格，采用简洁现代的界面设计
- 📱 **响应式布局**：支持移动端和桌面端
- 🚀 **高效性能**：使用React + Vite构建，加载速度快

## 技术栈

### 前端
- React 18
- Vite 5
- React Router 6
- CSS3 (Apple.com风格)

### 后端
- Express.js
- Node.js
- DeepSeek API (智能批改)

### 数据存储
- 内存存储 (开发环境)
- 支持MySQL (生产环境可配置)

## 项目结构

```
├── src/              # 前端代码
│   ├── App.jsx       # 主应用组件
│   ├── main.jsx      # 入口文件
│   └── index.css     # 样式文件
├── article/          # 题目和文章数据
│   ├── 题目&评分细则.json  # 题目数据
│   └── *.txt         # 文章内容文件
├── server.js         # 后端服务器
├── package.json      # 项目配置
└── vite.config.js    # Vite配置
```

## 安装步骤

1. **克隆项目**
2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置DeepSeek API**
   编辑 `server.js` 文件，将 `DEEPSEEK_API_KEY` 替换为实际的API密钥

4. **启动服务**
   - 启动后端服务器
     ```bash
     node server.js
     ```
   - 启动前端开发服务器
     ```bash
     npm run dev
     ```

5. **访问网站**
   打开浏览器访问 http://localhost:3000

## 使用方法

1. **浏览专题分类**：在首页查看按题目类型自动生成的分类
2. **选择练习题目**：点击分类查看对应类型的题目，或直接进入练习页面浏览所有题目
3. **阅读文章**：在题目详情页查看文章内容
4. **提交答案**：输入答案并提交，获得智能批改反馈
5. **查看批改结果**：系统会显示详细的评分和改进建议

## 数据说明

- **题目数据**：存储在 `article/题目&评分细则.json` 文件中
- **文章内容**：存储在 `article/` 目录下的 `.txt` 文件中，文件名对应题目中的出处
- **分类生成**：系统会自动从题目数据中提取类型作为分类

## 开发指南

### 前端开发
- 前端代码位于 `src/` 目录
- 使用 Vite 进行开发和构建
- 运行 `npm run dev` 启动开发服务器
- 运行 `npm run build` 构建生产版本

### 后端开发
- 后端代码位于 `server.js` 文件
- 支持内存存储和MySQL存储（可配置）
- 运行 `node server.js` 启动后端服务器
